### Host locally

Here's how to host the latest Fronvo site on your own:

```
git clone https://github.com/fronvo/server.git
cd server
npm i
```

Now fill out the required **[.env](https://github.com/Fronvo/server/blob/master/.env.example)** file

After that:

```
npm start
```
